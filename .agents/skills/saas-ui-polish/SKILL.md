# SaaS UI Polish

## Objetivo
Deixar o VestLink mais moderno, limpo e profissional.

## Quando Usar
Use em ajustes visuais amplos de interface SaaS sem mudar regras de negocio.

## Checklist
- Identificar telas prioritarias e arquivos de UI.
- Padronizar cards, botoes, formularios, tabelas e CTAs.
- Reduzir ruido visual e melhorar hierarquia.
- Conferir mobile, light/dark e contraste.

## Regras
- Mudancas incrementais e reversiveis.
- Nao alterar banco, auth, rotas ou fluxos.
- Preferir CSS existente e tokens do projeto.
