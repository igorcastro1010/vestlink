# Docs Cofre Sync

## Objetivo
Manter o cofre como contexto comprimido e confiavel do VestLink.

## Quando Usar
Use quando descobrir ou decidir algo estrutural e duravel no codigo.

## Checklist
- Abrir mapa em `cofre/vestlink/00 - Contexto/mapa-de-contexto.md`.
- Atualizar somente a nota relacionada.
- Registrar resumo, fontes e `revisado_em`.
- Nao duplicar codigo extenso.

## Regras
- Nunca registrar segredos, tokens ou dados pessoais.
- Nao atualizar cofre para ajustes cosmeticos triviais.
- Preferir notas curtas, densas e linkadas.
